2. 🛠 CORE BASE-CODE MODIFICATIONS (Why the bot wasn't working)
--------------------------------------------------------------
To adapt the original "Zero" repo for ASSS/HockeyZone, we patched the core:

A. ZoneController.h (The "Join Fix"):
   - Problem: VIE encryption prevents ?arena response; behaviors never loaded.
   - Fix: Moved CreateBehaviors() from ArenaNameEvent to JoinGameEvent.
   - Result: Bot "wakes up" immediately upon entering the arena.

B. BotController.cpp (The "Wipe Fix"):
   - Problem: Bot was wiping its own behavior memory during the join packet.
   - Fix: Commented out behaviors.Clear() in the JoinGameEvent handler.

C. main.cpp (The "Identity Fix"):
   - Problem: Server IP was not recognized as a specific "Zone."
   - Fix: Mapped the "Subgame" server entry to the HockeyZoneController.


3. 🧠 ML PIPELINE (The "ML" Tools)
--------------------------------------
We synchronized the Python training tools with the C++ Bot's coordinate system:

- convert.py  : Now extracts Ball Position (BX, BY) from .rec files.
- prepare2.py : Vectorized processing. Converts raw pixels to TILE units (/16.0).
- train2.py   : Trains the Neural Net (7 inputs -> 2 outputs [DirX, DirY]).
- onnx2c      : Converts the .onnx model to pure C code for zero-dependency speed.
